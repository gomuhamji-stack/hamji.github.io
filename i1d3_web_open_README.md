주요 커맨드 설명 및 실험 팁
초기화 (Identity - 49 'I'):

센서가 살아있는지 확인하는 가장 기본적인 명령입니다. 응답으로 69 31 44 69... (i1Display...) 같은 아스키 코드가 돌아오면 성공입니다.

Unlock (4F 45 4D 20 49 44 'OEM ID'):

일부 OEM 버전 센서는 특정 키를 보내야 측정이 활성화됩니다. 위 커맨드는 일반적인 잠금 해제 문자열입니다. 장치가 ACK를 보낼 때까지 값을 바꿔가며 타이핑 실험을 할 수 있습니다.

Dark Calibration (44 'D'):

센서를 빛이 차단된 상태(또는 렌즈 캡을 닫은 상태)에서 영점 조절을 수행합니다. 측정 전 필수 단계입니다.

Color Value Read (4D 'M'):

실제 측정 명령입니다. 이 명령 후 장치는 측정을 수행하고 결과 데이터를 보냅니다. 결과값은 단순히 숫자가 아니라 센서 내부의 Raw Count 값이므로, 이를 의미 있는 XYZ나 RGB로 바꾸려면 복잡한 계산식(Matrix 연산)이 필요하지만, 위 인터페이스를 통해 데이터가 변하는지는 즉시 확인 가능합니다.

이 방식의 효율성
직관적 디버깅: 코드를 수정하고 새로고침할 필요 없이, 에디트 창에서 4D 뒤에 01이나 02 등을 붙여보며 장치가 어떻게 반응하는지 즉각 테스트할 수 있습니다.

커스텀 실험: 잘 안될 경우, 로그에 찍히는 수신 데이터를 보고 다음 커맨드를 무엇을 보낼지 결정하는 "대화형" 실험이 가능하여 개발 속도가 매우 빠릅니다.
